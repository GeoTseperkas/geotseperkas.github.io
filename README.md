## 📖 About

**George Tseperkas Portfolio** is a central hub for my professional identity:
1.  **Background & Skills:** Information about my journey and technical capabilities.
2.  **Project Showcase:** A curated display of my work and side projects.

It features a clean, responsive design with smooth animations, serving as a testament to my commitment to continuous improvement as a developer.

## 🛠 Tech Stack

* **Core:** React 18, TypeScript, Vite
* **Styling:** Vanilla CSS
* **Animations:** Framer Motion
* **Icons:** React Icons

## 🚀 Quick Start

1.  **Clone the repo**
    ```bash
    git clone https://github.com/geotseperkas/geotseperkas.github.io.git
    cd geotseperkas.github.io
    ```

2.  **Install dependencies**
    ```bash
    npm install
    ```

3.  **Run locally**
    ```bash
    npm run dev
    ```

## 🎨 Key Features

* **Modern Dark Theme:** A minimalist, high-contrast aesthetic that is easy on the eyes.
* **Component-Based:** Modular architecture for easy maintenance and scaling.
* **Smooth Animations:** Integrated page transitions and micro-interactions.
* **Fully Responsive:** Fluid layouts that look great on any device.
